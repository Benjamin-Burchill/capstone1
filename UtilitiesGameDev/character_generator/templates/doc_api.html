Usage Examples

from character_generator_advanced import AdvancedCharacterGenerator # Initialize generator generator = AdvancedCharacterGenerator() # Load preset generator.load_preset('human') # Adjust parameters generator.set_parameter('height', 1.2) generator.set_parameter('muscle_definition', 0.5) generator.set_parameter('jaw_width', 0.3) # Generate mesh mesh = generator.generate() # Export mesh.export('my_character.obj')

Core Classes

  • AdvancedCharacterGenerator - Main generator class with parameter management
  • AdvancedCharacterMesh - Mesh container with morphing capabilities
  • CharacterParameters - Parameter container with validation
  • SymmetricHumanoidBuilder - Creates perfectly symmetric base meshes
  • AdvancedMorphingSystem - Applies complex deformations to meshes
