Common Workflows

Creating a Fantasy Race

Goal: Create a unique fantasy race for your game

  1. Start with a base preset (Human or similar)
  2. Adjust global parameters:
    • Height for race size
    • Build for body type
    • Muscle definition for physique
  3. Modify facial features:
    • Eye size and spacing
    • Jaw and chin proportions
    • Nose characteristics
  4. Add special features:
    • Horns for demonic races
    • Elongated ears for elven types
    • Pronounced brow ridge for primitive races
  5. Save as a new preset for consistency

Creating Character Variations

Goal: Generate multiple variations of the same character type

  1. Load your base preset
  2. Use the Randomize button with constraints:
    • Lock important parameters
    • Set variation ranges
  3. Fine-tune each variation manually
  4. Export each variation with descriptive names
  5. Create a catalog of your variations

Optimization for Games

Goal: Prepare characters for real-time rendering

  1. Choose appropriate quality level:
    • Low (~316 vertices) for mobile/background characters
    • Medium (~442 vertices) for main characters
    • High (~2270 vertices) for hero characters
  2. Export as OBJ or STL format
  3. Post-process in your 3D software:
    • Apply texture coordinates
    • Add rigging/skeleton
    • Create LODs if needed
  4. Import into your game engine

Tips & Tricks

ðŸ’¡ Pro Tips

  • Start with presets: Don't create from scratch - modify existing presets
  • Small changes: Subtle parameter adjustments often look more natural
  • Test in-game: Always preview characters in your target environment
  • Save incrementally: Keep versions as you work
  • Use symmetry: The system guarantees perfect bilateral symmetry
