Parameter Guidelines

Value Ranges

Most parameters use a normalized range of -1.0 to 1.0 where:

  • -1.0 = Minimum/smallest value
  • 0.0 = Default/neutral value
  • 1.0 = Maximum/largest value

Exception: Height parameter uses 0.5 to 1.5 as a multiplier of base height.

Global vs Regional Parameters

  • Global Parameters (height, build, muscle_definition) affect the entire character
  • Regional Parameters target specific body parts for fine control
  • Parameters work together - adjust multiple parameters for best results

Special Parameters

Fantasy features for creating non-human characters:

  • horn_size - Add horns to the character
  • horn_position - Control horn placement
  • tail_length - Add a tail (not yet implemented)
  • tail_thickness - Control tail size (not yet implemented)
